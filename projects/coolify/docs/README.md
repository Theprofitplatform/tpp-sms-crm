## 📞 Support

**Troubleshooting**:
1. Check [Complete SEO Platform Guide](guides/COMPLETE-SEO-PLATFORM-GUIDE.md)
2. Run health check: `./scripts/run-all-automation.sh health`
3. View logs: `./scripts/run-all-automation.sh logs`
4. Review [Deployment Issue Analysis](deployment/DEPLOYMENT-ISSUE-ANALYSIS.md)

**Common Issues**:
- **Service Down**: Check `docker ps` and restart with `docker-compose restart`
- **Domain Not Working**: Verify Coolify proxy configuration
- **SSL Issues**: Wait 2-3 minutes for Let's Encrypt certificate generation
