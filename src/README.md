# Source Code

Main application source code organized by feature.

## Structure

- `api/` - External API clients (WordPress REST API, SerpAPI, etc.)
- `audit/` - SEO audit engines and schema validation
- `monitoring/` - Ranking and performance monitoring
- `deployment/` - Deployment automation and WordPress integration
- `utils/` - Shared utilities, helpers, and emergency fixes

## Entry Points

- Main application: `src/index.js` (to be created)
- Monitoring: `src/monitoring/monitor-rankings.js`
- Deployment: `src/deployment/deploy-*.js`
