## 💡 Use Cases

### 1. Agency Portfolio Management

**Scenario:** Manage 20+ clients and identify which need attention.

```bash
# Get full comparison
curl -X POST http://localhost:9000/api/local-seo/compare \
  -H "Content-Type: application/json" \
  -d '{}'

# Check distribution
curl http://localhost:9000/api/local-seo/distribution

# Get critical alerts
curl "http://localhost:9000/api/local-seo/alerts?severity=critical"
```

**Actions:**
- Focus on clients in "needsAttention" list
- Prioritize opportunities by impact/effort
- Monitor alerts for proactive management

### 2. Client Benchmarking

**Scenario:** Show client how they compare to portfolio average.

```bash
# Compare client to all others
curl -X POST http://localhost:9000/api/local-seo/compare \
  -H "Content-Type: application/json" \
  -d '{"clientIds": ["yourclient"]}'

# Get specific comparisons
curl http://localhost:9000/api/local-seo/compare/yourclient/competitor
```

**Use in Reports:**
- "You rank #3 out of 20 clients"
- "Portfolio average is 72, you're at 85 (+13)"
- "Top performer is at 95 (+10 points ahead)"

### 3. Proactive Monitoring

**Scenario:** Get notified when scores drop.

```bash
# Set aggressive thresholds
curl -X POST http://localhost:9000/api/local-seo/alerts/thresholds \
  -H "Content-Type: application/json" \
  -d '{"scoreDrop": 5, "warningScore": 70}'

# Check alerts daily
curl http://localhost:9000/api/local-seo/alerts
```

**Automation:**
- Run scheduled audits daily
- Check alerts endpoint
- Send summary email if critical alerts exist

### 4. Opportunity Identification

**Scenario:** Find quick wins across portfolio.

```bash
# Get all opportunities
curl http://localhost:9000/api/local-seo/opportunities

# Filter for high-impact, low-effort
jq '.opportunities.opportunities[] | select(.opportunities[].impact == "high" and .opportunities[].effort == "low")'
```

**Priority Actions:**
- Schema markup additions (low effort, medium impact)
- NAP fixes (medium effort, high impact)
- Citation building (high effort, medium impact)

## 🔧 Integration Examples

### Daily Monitoring Script

```bash
#!/bin/bash
# daily-local-seo-check.sh

echo "=== Daily Local SEO Check ==="

# Get alerts
CRITICAL=$(curl -s "http://localhost:9000/api/local-seo/alerts?severity=critical" | jq '.alerts | length')
WARNING=$(curl -s "http://localhost:9000/api/local-seo/alerts?severity=warning" | jq '.alerts | length')

echo "Critical Alerts: $CRITICAL"
echo "Warning Alerts: $WARNING"

if [ $CRITICAL -gt 0 ]; then
    echo "⚠️  CRITICAL ALERTS NEED ATTENTION!"
    curl -s "http://localhost:9000/api/local-seo/alerts?severity=critical" | jq '.alerts[]'
fi

# Get opportunities
OPPS=$(curl -s "http://localhost:9000/api/local-seo/opportunities" | jq '.opportunities.totalOpportunities')
echo "Total Opportunities: $OPPS"

# Get distribution
curl -s "http://localhost:9000/api/local-seo/distribution" | jq '.distribution.percentages'
```

### Weekly Report Generation

```bash
#!/bin/bash
# weekly-report.sh

DATE=$(date +%Y-%m-%d)
REPORT="local-seo-report-$DATE.txt"

echo "Local SEO Weekly Report - $DATE" > $REPORT
echo "======================================" >> $REPORT
echo "" >> $REPORT

# Get comparison
curl -s -X POST http://localhost:9000/api/local-seo/compare \
  -H "Content-Type: application/json" \
  -d '{}' | jq . >> $REPORT

# Email report
mail -s "Local SEO Weekly Report" your-email@example.com < $REPORT
```
