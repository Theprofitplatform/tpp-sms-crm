## 🎯 Usage

### **NPM Scripts (Recommended)**

```bash
# Test
npm run autofix:test               # Run test suite

# Preview
npm run autofix:dry-run            # See what would change

# Execute
npm run autofix:run                # Sequential (12 min)
npm run autofix:parallel           # Parallel (4 min) ⭐ RECOMMENDED
npm run autofix:ai                 # AI + Parallel (4-5 min)
npm run autofix:queue              # Background (instant)

# Monitor
npm run cache:stats                # Cache statistics
npm run queue:stats                # Queue statistics

# Maintain
npm run cache:flush                # Clear cache
npm run queue:clean                # Clean old jobs
```

### **Direct Commands**

```bash
# Basic
node auto-fix-all-upgraded.js --client=CLIENT_ID

# With options
node auto-fix-all-upgraded.js --parallel --ai --client=CLIENT_ID

# Specific client
node auto-fix-all-upgraded.js --parallel --client=freedomactivewear
```

### **Command Line Flags**

- `--dry-run` - Preview without applying changes
- `--parallel` - Run 3 engines simultaneously (3x faster)
- `--ai` - Enable AI-powered suggestions
- `--queue` - Use job queue (background processing)
- `--client=ID` - Specify client ID

## 🔧 Troubleshooting

### **Tests Fail**

```bash
# Run test suite
npm run autofix:test

# Common fixes:
# - Redis not installed → Skip Redis tests (OK)
# - OpenAI key missing → Skip AI tests (OK)
# - WordPress connection → Check clients/*.env
```

### **Redis Errors**

```bash
# Check if Redis is running
redis-cli ping

# Start Redis
redis-server &

# Or run without Redis (will work fine)
npm run autofix:parallel
```

### **OpenAI Errors**

```bash
# Set API key
export OPENAI_API_KEY=sk-...

# Or run without AI
npm run autofix:parallel
```

### **Performance Issues**

```bash
# Use parallel mode
npm run autofix:parallel

# Or queue-based
npm run autofix:queue

# Increase memory if needed
NODE_OPTIONS="--max-old-space-size=4096" npm run autofix:parallel
```
