## 🔧 TROUBLESHOOTING GUIDE

### Problem: "401 Unauthorized" Error

**Cause:** Wrong credentials or insufficient permissions

**Fix:**
1. Verify WordPress username in `.env` file
2. Check app password has no typos or spaces
3. Ensure user role is Administrator or Editor
4. Try regenerating the app password
5. Test with: `node test-auth.js`

---

### Problem: "Cannot read property 'data'" Error

**Cause:** WordPress site unreachable or URL wrong

**Fix:**
1. Verify `WORDPRESS_URL` in `.env` is correct
2. Open URL in browser to confirm site is online
3. Check for www vs non-www redirects
4. Ensure `https://` is included

---

### Problem: Client Skipped During Batch Processing

**Cause:** Client not marked as "active" or missing .env file

**Fix:**
1. Check `clients/clients-config.json` → `"status": "active"`
2. Verify `.env` file exists: `ls -la clients/[client-id].env`
3. Ensure client-id matches exactly (case-sensitive)

---

### Problem: Reports Not Generating

**Cause:** Logs directory doesn't exist or permissions issue

**Fix:**
1. Create directory: `mkdir -p logs/clients/[client-id]`
2. Check write permissions
3. Verify disk space available

---

### Problem: Optimization Takes Too Long

**Cause:** Large number of posts or slow connection

**Solution:**
- Normal: 2-3 minutes for 50 posts
- Slow: 5-10 minutes for 200+ posts
- If over 10 minutes, check internet connection
