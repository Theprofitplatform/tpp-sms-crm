## 🛠️ YOUR AUTOMATION TOOLKIT

### **Core Scripts**

#### 1. **auto-fix-all.js** - Master Orchestrator
Run all SEO fixes at once
```bash
node auto-fix-all.js
```

**What it does:**
- Fixes short titles (adds branding)
- Converts duplicate H1s to H2
- Adds missing image alt text
- Generates comprehensive report

**When to use:** Weekly optimization

---

#### 2. **auto-fix-titles.js** - Title Optimizer
Fix titles only
```bash
node auto-fix-titles.js          # Live mode
node auto-fix-titles.js --dry-run  # Preview mode
```

**What it does:**
- Analyzes all post titles
- Adds branding to short titles
- Optimizes length to 30-60 chars

**When to use:** After publishing new posts

---

#### 3. **auto-fix-h1-tags.js** - H1 Fixer
Fix heading structure
```bash
node auto-fix-h1-tags.js          # Live mode
node auto-fix-h1-tags.js --dry-run  # Preview mode
```

**What it does:**
- Detects multiple H1 tags
- Keeps first H1, converts others to H2
- Improves content hierarchy

**When to use:** Theme changes or new content

---

#### 4. **auto-fix-image-alt.js** - Image Alt Text
Add alt attributes
```bash
node auto-fix-image-alt.js          # Live mode
node auto-fix-image-alt.js --dry-run  # Preview mode
```

**What it does:**
- Scans all images
- Generates descriptive alt text
- Improves accessibility & SEO

**When to use:** After adding new images
