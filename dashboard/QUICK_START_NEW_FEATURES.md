## 1️⃣ Position Tracking CSV Analysis

### Quick Start (30 seconds)

```bash
1. Navigate to "Position Tracking" in sidebar
2. Click "Load Demo Data" button
3. Explore the 5 tabs with sample analysis
```

### Upload Your Own Data (1 minute)

```bash
1. Export CSV from SEMrush or Ahrefs
2. Drag & drop into the upload area
3. Analysis runs automatically
4. View results across 5 tabs
```

### What You'll See

**Stats Cards (Top of Page):**
- Total Keywords
- Top 10 Rankings
- Improved Positions
- Declined Positions
- Quick Win Opportunities

**Critical Issues Alert (If Any):**
- Red alert box showing urgent problems
- Keywords that lost significant positions
- Impact level (CRITICAL/HIGH/MEDIUM/LOW)
- Immediate action required

**5 Analysis Tabs:**

1. **Quick Wins** - Positions 11-20 with traffic potential
   - Shows current traffic vs. potential
   - Estimated monthly traffic gain
   - Priority level for each keyword

2. **Top Performers** - Your best rankings (positions 1-10)
   - Track your winners
   - Monitor for any declines
   - Maintain these positions

3. **Declines** - Keywords losing positions
   - Impact assessment
   - Recommended recovery actions
   - Prioritized by urgency

4. **AI Overview** - Google AI feature tracking
   - Keywords appearing in AI results
   - New opportunity for visibility
   - SERP features displayed

5. **All Keywords** - Complete dataset
   - Full CSV data analyzed
   - Position changes tracked
   - Volume data included

### Export Your Analysis

```bash
Click "Export Analysis" button
- Downloads CSV with insights
- Includes traffic estimates
- Shows priority levels
- Ready for client reports
```

## 3️⃣ Terminal Output View

### Quick Start (5 seconds)

```bash
1. Navigate to "Control Center"
2. Change "View" dropdown to "Terminal"
3. Run any operation
4. Watch real-time output
```

### What You'll See

```terminal
[10:23:45] ▸ Starting audit for client: Hot Tyres
[10:23:47] ✓ audit completed successfully for Hot Tyres
[10:24:12] ⚠ Stopping job: optimization (Client X)
[10:24:13] ✓ Job stopped: optimization
[10:24:45] ✗ audit failed for Client Y: Connection timeout
```

### Features

**Color Coding:**
- 🔵 Blue (Info) - General information
- 🟢 Green (Success) - Completed successfully
- 🟡 Yellow (Warning) - Important notices
- 🔴 Red (Error) - Failed operations

**Filtering:**
```bash
- Click "All" - Show everything
- Click "Errors" - Show only errors
- Click "Warnings" - Show only warnings
```

**Actions:**
```bash
- Click "Copy" - Copy to clipboard
- Click "Download" - Save as .txt file
- Click "Clear" - Reset terminal
```

### Use Cases

**Debugging:**
```bash
1. Switch to Terminal view
2. Run the failing operation
3. See exact error message
4. Copy error for troubleshooting
```

**Monitoring:**
```bash
1. Keep Terminal view open
2. Run multiple operations
3. Watch real-time progress
4. Filter errors when needed
```

**Auditing:**
```bash
1. Run all operations for the day
2. Download terminal log
3. Review what happened
4. Share with team/client
```

## 🆘 Troubleshooting

### Position Tracking

**CSV Upload Fails:**
- Check file is .csv format
- Ensure columns include: Keyword, Position
- Try demo data to test feature
- Maximum file size: 50MB

**No Quick Wins Found:**
- Normal if no keywords in positions 11-20
- Focus on other tabs
- Try different time period

**Traffic Estimates Seem Wrong:**
- Based on industry CTR averages
- Adjust for your specific niche
- Use as relative comparison

### GSC Enhanced

**Sync Not Working:**
- Complete setup wizard first
- Check GSC credentials
- Verify API enabled
- Test connection in settings

**No Opportunities:**
- Normal for new sites
- Continue building content
- Check back weekly

**Recommendations Not Showing:**
- Click "View" button on keywords
- Ensure data synced
- Try different category tabs

### Terminal View

**No Output:**
- Ensure view set to "Terminal"
- Run an operation
- Wait for completion
- Check "All" filter is selected

**Output Not Color-Coded:**
- Normal for some operations
- Colors added automatically
- Depends on message type
