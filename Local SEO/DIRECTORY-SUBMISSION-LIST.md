## 🚫 WHAT TO AVOID

❌ Using different NAP formats (huge mistake!)
❌ Skipping verification emails (listings won't go live)
❌ Using P.O. Box addresses (use physical address)
❌ Keyword stuffing business name ("Best #1 Sydney Car Buyer...")
❌ Leaving fields blank (complete all available fields)
❌ Using inconsistent phone numbers (same number everywhere)
❌ Forgetting to save login credentials (you'll need them later)
❌ Submitting to spam directories (stick to this list)
❌ Giving up after first 5 (the compound effect happens at 20+)
